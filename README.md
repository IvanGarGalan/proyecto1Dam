proyecto1Dam

Proyecto de creacion de aplicacion funcional en terminal usando Dart para la asignatura Proyecto de 1º de Desarrollo de Aplicaciones Multiplataforma

Esta aplicación se ha creado a base de los siguientes requisitos:

1.Debe crear una base de datos(la propia aplicación)* e interactuar con ella.(Recomendable que una de las interacciones sea un login,pues se va a ver en clase).

2.Interactuar con una API.(Preferiblemente de acceso libre,sin apikey ni accesos https://github.com/public-apis/public-apis)

3.La aplicación debe usar la Programación orientada a Objetos.

4.Ejecución multiple(La apliación no puede hacer siempre lo mismo,poner varias opciones,con más de un menú).

5.Puede ser un juego, una app de gestión,etc…

6.Se debe hacer una presentación de 5-10 minutos.
